📚 DeBuDDi Guide Zone!

Welcome, Buddy! This is your step-by-step guide to joining the fun, contributing, and making magic happen in our community. 🎉

🚀 Quick Start

  • Clone the repo:
    git clone https://github.com/DeBuDDi/debuddi.github.io.git
  • Install dependencies:
    npm install
  • Run locally:
    npm start

🤝 How to Contribute

  • Fork the repo and make your changes
  • Send a Pull Request—bonus points for adding a meme in your PR description! 😆
  • Check out open issues or suggest new ones!

🦸‍♂️ Need Help?

  • Ask for help in our Discord
  • Check the Support page for FAQs and contact info
  • Or just open an issue on GitHub!

🌟 Stay Connected

  • Subscribe to our YouTube channel
  • Share the project, invite your friends, and don't forget to ⭐️ us!

Thanks for being a Buddy! 🎈
